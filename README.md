# Be Grace — からだ診断 × 運動セルフケア会員サイト

> 産後・育児中の女性が、自分のからだに優しく戻ってこられるための、運動セルフケア・スタジオ。
> 「治す」ではなく「ひらく」、「我慢」ではなく「受け取る」。

このリポジトリは、谷尾尚子様（整骨院／産後ケア指導）向けに構築された **デモ用 Web プロダクト** です。
投資家・顧客・パートナー向けの「触れる完成品」として、フロントエンドのみで完結する高品質プロトタイプとして実装されています。

## 構成

- `web/` — Next.js 16 + TypeScript + Tailwind CSS v4 + Framer Motion + Zustand（フロントエンドのみ）
- `vercel.json` — Vercel デプロイ設定（`web/` を rootDirectory 扱い）

## 主な機能

### 公開（マーケティング）

- **ランディング**: 「理想の未来を、受け取る、からだ診断」
- **診断コレクション**: 骨格5タイプ／セルフケア5タイプ／AI 姿勢診断
- **個別相談予約 / 既存講座 / About**

### 診断アプリ

- 骨格5タイプ診断（7問）
- セルフケア5タイプ診断（6問）
- AI 姿勢診断モック（写真3枚アップロード → 進捗演出 → スコアレポート）

### 会員サイト（本丸）

- ダッシュボード（ポイント／レベル／連続日数）
- 今日の記録（カテゴリ・強度・気分の入力 + ポイント加算）
- ゴール（作成・進捗・ご褒美管理）
- ポイント（ご褒美交換 / 履歴）
- 動画ライブラリ（VIP コンテンツ表示分岐）
- 動画詳細（プレビュー → 完了で記録自動追加）
- 相談（先生 + AI のスレッド／返信モック）
- 管理者画面（権限切替で表示／会員管理／動画コンテンツ／診断テンプレート）

### 権限（モック）

サイドバーの「Demo: 権限切替」で **一般 / VIP / 管理** を切り替え可能。
表示される動画・解放される機能・管理者画面の出現が変化します。

## ローカル起動

```bash
cd web
pnpm install --no-frozen-lockfile
pnpm next dev
# → http://localhost:3000
```

## デプロイ

GitHub に push 後、Vercel が自動でビルド・公開します。
本番 URL は Vercel ダッシュボードを参照。

## デザイン哲学

ジョナサン・アイブが整え、スティーブ・ジョブズがレビューしたような、繊細な余白・上品なローズゴールド・優しい曲線。

- カラー: クリーミーホワイト / ローズゴールド / ディープローズ
- タイポ: Cormorant Garamond (display) + Noto Serif JP + Noto Sans JP
- アニメ: Framer Motion + cubic-bezier(0.22, 1, 0.36, 1)
